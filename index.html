Slash Commands

Simple. Powerful. Familiar.

All commands use Discord's native slash command system — just type / and Catalyst appears.

/
ban
Ban a member from the server with an optional reason and duration.
/
kick
Remove a member from the server. They can rejoin with an invite link.
/
mute
Timeout a member for a specified duration. Great for cooling things down.
/
warn
Issue a formal warning to a member and log it to their record.
/
play
Play a song or playlist from YouTube or Spotify in a voice channel.
/
stats
Display server statistics, member counts, and activity information.
/
giveaway
Start a giveaway in any channel. Set duration, prize, and winner count.
/
setup
Run the setup wizard to configure Catalyst for your server in minutes.
